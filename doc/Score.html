    calculScore() click to toggle source

    Cette méthode calcule le nombre de points d'un score.

    # File Classement/Score.rb, line 45
def calculScore()
        tMax = 0
        case @difficulte
                when "easy"
                        tMax = 60
                when "normal"
                        tMax = 90
                when "hard"
                        tMax = 120
        end
        temp = (tMax / @temps) *100
        if(temp < 0) then temp = 0 end
        @points = (temp*temp)%1000000
end
